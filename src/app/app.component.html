Random Data API

Get random fake data from https://random-data-api.com/

  • Name: {{name}}

    GET /api/name/random_name

  • Vehicle: {{vehicle}}

    GET /api/vehicle/random_vehicle

  • Food: {{food}}

    GET /api/food/random_food 

ngsw-config.json
{
  "dataGroups": [
    {
      "name": "api-freshness",
      "urls": [
        "https://random-data-api.com/api/vehicle/random_vehicle"
      ],
      "cacheConfig": {
        "maxSize": 0,
        "maxAge": "0u",
        "strategy": "freshness"
      }
    },
    {
      "name": "api-performance",
      "urls": [
        "https://random-data-api.com/api/food/random_food"
      ],
      "cacheConfig": {
        "strategy": "performance",
        "maxSize": 100,
        "maxAge": "2d"
      }
    }
  ]
}
