Random Data API

Get random fake data from https://random-data-api.com/

  • Name: {{name}}

    GET /api/name/random_name
    ngsw-config no configured

  • Vehicle: {{vehicle}}

    GET /api/vehicle/random_vehicle
    ngsw-config freshness strategy

  • Food: {{food}}

    GET /api/food/random_food
    ngsw-config performance strategy 

ngsw-config.json
{
  "dataGroups": [
    {
      "name": "api-freshness",
      "urls": [
        "https://random-data-api.com/api/vehicle/random_vehicle"
      ],
      "cacheConfig": {
        "strategy": "freshness",
        "maxSize": 100,
        "maxAge": "5m",
        "timeout": "1s"
      }
    },
    {
      "name": "api-performance",
      "urls": [
        "https://random-data-api.com/api/food/random_food"
      ],
      "cacheConfig": {
        "strategy": "performance",
        "maxSize": 100,
        "maxAge": "5m"
      }
    }
  ]
}
